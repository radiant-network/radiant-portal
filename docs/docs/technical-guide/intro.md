# Technical Guide

Welcome to the technical guide for the Radiant project. 

This guide will help you:
- [Install](deployment/intro.md) and [configure](administration/configuration.md) Radiant for local, staging, or production environments
- Understand Radiant's [data model](data-model/model.md) and [architecture](architecture/overview.mdx)
- Use and extend Radiant's [API](api/overview.mdx) for building integrations and custom applications
- [Develop, test, and contribute](developers/contribute.md) new features

Whether you're responsible for deployment, configuration, or development, this guide will equip you with the knowledge to work effectively with Radiant and leverage its full capabilities.
