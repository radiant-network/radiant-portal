# upload-id-facet

Upload-list facet used to bulk-load identifiers (genes, etc.) into the active query. Rendered as a button that opens a dialog. Used for `FilterTypes.UPLOAD_LIST` aggregations.

# Config

```json
{
  key: 'upload_gene_list',
  translation_key: 'upload_gene_list',
  type: FilterTypes.UPLOAD_LIST,
}
```
