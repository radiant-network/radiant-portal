# search-query-pill

Pill rendered for an [IValueFacet](../type/IValueFacet.md) coming from a [search-facet](../facets/search-facet.md). Built by [query-bar](../query-bar.md).

Unlike other pills, no popover edition (search facet is async-only).

## Props

```typescript
type SearchPillProps = {
  sqon: IValueFacet;
};
```
